## Vaccination Data Analysis & VisualizationTechnologies 

### Used: Python, SQL, Power BI, Exploratory Data Analysis (EDA)
- Domain: Public Health & Epidemiology Engineered an end-to-end data analysis pipeline: 
1. Developed a comprehensive solution to analyze global vaccination coverage, assessing vaccine effectiveness and supporting targeted resource allocation for public health agencies.
2. Developed robust Python data cleaning scripts: Handled missing data imputation and normalized disparate epidemiological datasets, ensuring strict unit consistency and uniform date formatting across global records.
3. Architected a normalized SQL database: Designed and populated relational tables for vaccination, incidence, and schedule data, implementing primary and foreign keys to ensure referential integrity and optimize query performance.
4. Built interactive Power BI dashboards: Created dynamic business intelligence reports integrating SQL data via scheduled refreshes to visualize vaccination rates and disease incidence over time.
5. Delivered actionable public health insights: Designed geographical heatmaps, trend lines, and KPI indicators that enabled stakeholders to identify low-coverage areas, forecast vaccine demand, and formulate data-driven policies.
